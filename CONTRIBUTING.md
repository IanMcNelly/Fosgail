# Contributing to Fosgail

First off, thank you for considering contributing to Fosgail! 

## Development Setup

1. Fork and clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.

## Making Changes

1. Create a new branch for your feature or bugfix: `git checkout -b feature-name`
2. Make your changes in the `src` directory.
3. Ensure your code is clean and properly formatted.
4. Test your changes locally.

## Submitting a Pull Request

1. Push your branch to your fork.
2. Open a Pull Request against the `main` branch of this repository.
3. Provide a clear description of the problem you are solving and how your changes fix it.
4. Include screenshots if your PR introduces visual UI changes.

We will review your PR as soon as possible!

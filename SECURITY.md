# Security Policy

## Supported Versions

Currently, only the latest release of Fosgail is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.2.x   | :white_check_mark: |
| < 1.2   | :x:                |

## Reporting a Vulnerability

We take the security of Fosgail seriously. If you discover a vulnerability, please do **not** open a public issue.

Instead, please report it via the GitHub Security Advisory feature in this repository.

1. Go to the [Security Advisories page](https://github.com/IanMcNelly/markdown-viewer/security/advisories) of the repository.
2. Click on **Report a vulnerability**.
3. Provide a detailed description of the issue, steps to reproduce it, and any potential mitigations you suggest.

We will review the submission and, if it is confirmed, work to release a fix as soon as possible.
